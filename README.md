# Math

The Math library allows for some basic math calculations.	


## Features
* Math Constants like pie
* Geometric shapes like lines

## Compilation Instructions

* Debian Linux
    1. sudo  mkdir build
    2. cd build
    3. cmake ../ [-DBUILD_SHARED_LIBS="True"][-DCMAKE_BUILD_TYPE="True"]
    4. make
    5. sudo make install
